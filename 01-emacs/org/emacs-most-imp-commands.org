\* BookMarks


1. C-x r m RET : Set the bookmark for the visited file, at point.
2. C-x r m bookmark RET : Set the bookmark named bookmark at point (bookmark-set).
3. C-x r b bookmark RET : Jump to the bookmark named bookmark (bookmark-jump).
3. C-x r l : List all bookmarks (list-bookmarks).
4. M-x bookmark-save : Save all the current bookmark values in the default bookmark file. 
6. M-x bookmark-delete RET bookmark RET : Delete bookmark named bookmark
